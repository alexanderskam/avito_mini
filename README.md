Веб-приложение — личный кабинет продавца с AI-ассистентом для улучшения объявлений.

Приложение позволяет:

управлять списком объявлений
просматривать и редактировать карточки товаров
получать рекомендации от AI для улучшения описания и цены
🚀 Стек технологий
Frontend
React 18 + TypeScript
react-router-dom
Zustand / Redux (зависит от реализации)
TanStack Query (React Query) / Axios
UI-библиотека (например, Material UI / Ant Design)
Vite / Webpack
Backend
Node.js v20+
Предоставленный API (порт 8080)
AI
Ollama (локально, модель llama3)
или
Внешний LLM API (например, Grok / OpenAI-совместимый)
📦 Функциональность
📄 1. Список объявлений (/ads)
Отображение всех объявлений
Поиск по названию
Сортировка (по дате, цене)
Фильтрация:
по категории
только требующие доработки
Пагинация (10 объявлений на страницу)
🔍 2. Просмотр объявления (/ads/:id)
Полная информация о товаре
Характеристики в зависимости от категории
Блок "Требуются доработки"
Переход к редактированию
✏️ 3. Редактирование объявления (/ads/:id/edit)
Форма редактирования:
категория, название, цена
характеристики
описание
AI-функции:
✨ Улучшение описания
💰 Определение рыночной цены
Сохранение:
API запрос (PUT)
автосохранение в localStorage
Отмена изменений
⭐ Дополнительно
AI-чат с контекстом объявления
Сравнение "до / после"
Тёмная тема
Отмена запросов при переходе между страницами
